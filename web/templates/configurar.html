❓ Ayuda

1. ¿Cómo obtengo mi API Key de API-Sports?

  1. Ve a dashboard.api-football.com
  2. Regístrate o inicia sesión
  3. En el dashboard, encontrarás tu API key
  4. El plan gratuito incluye 100 requests/día

2. ¿Cómo genero una Contraseña de Aplicación en Gmail?

  1. Ve a myaccount.google.com/apppasswords
  2. Inicia sesión con tu cuenta de Gmail
  3. Si no tienes activada la verificación en 2 pasos, actívala primero
  4. Selecciona "Otra (nombre personalizado)" y ponle un nombre como "Sistema Apuestas"
  5. Google generará una contraseña de 16 caracteres
  6. Copia esa contraseña y pégala aquí

3. ¿Es seguro?

Sí. Las credenciales se guardan en un archivo .env en el servidor y nunca se comparten. La contraseña de aplicación solo permite enviar emails, no da acceso completo a tu cuenta de Gmail.
